# makelaardij-notify
Apartment scraper with simple UI for Rotterdam real-estate brokers

### Features

- Filter on price, living area, construction year and availability
- Push notifications for your filter
- 10+ custom scrapers
- Dark mode


### Stack

##### Backend

- Docker
- FastAPI
- MongoDB
- BeautifulSoup

##### Frontend

- React
- Typescript
- MobX
- Material-UI
